# Setup
## Install libraries
### Through conda

```
conda install --file requirements.txt
```

### Through pip

```
pip install -r requirements.txt
```

## Install ffmpeg (required by pydub) 

### from conda-forge
```
conda install -c conda-forge ffmpeg --yes
```
### On macOS, you can alternatively install ffmpeg via Homebrew
```
brew install ffmpeg
```
